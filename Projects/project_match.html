ProjectMatch ✨

A smart, aesthetic, and automated university project allocation system. Say goodbye to messy spreadsheets! Designed to match students to final-year academic projects with fairness, speed, and style.

  • Role Full Stack Developer
  • Scope Django Application Build
  • Timeline Final Year Project
Python Django Bootstrap SQLite

Premium Aesthetic

Built with the official Aston University color palette, featuring deep purples, soft gradients, and interactive push-buttons for a satisfying UX. Beautifully designed for Students, Supervisors, and Admins.

Drag-and-Drop Magic

Students can effortlessly select and rank their top 3 project preferences using our tactile, engaging, and interactive drag-and-drop user interface.

The Fairness Engine

Our custom algorithmic matching engine calculates profile suitability and strict project quotas to allocate the entire cohort automatically, quickly resolving academic mismatches!

Dashboards & Visuals

Dynamic Chart.js visuals and progress bars give supervisors real-time insights into student prerequisite matches. A friendly navbar bell alerts users of new actions alongside SweetAlert2 modals!

Setup & Testing

Want to take ProjectMatch for a spin? We use pipenv to manage our Python magic. 🪄

Prerequisites

  • Python 3.x installed on your machine.
  • Pipenv installed (pip install pipenv).

Installation

  • git clone https://github.com/oyin59/Project-match.git
  • pipenv install then pipenv shell to start the environment.
  • python manage.py migrate to set up the SQLite database.
  • Highly Recommended: python manage.py seed_final_data to beautifully generate dummy data (Students, Supervisors, Admins, Projects).
  • python manage.py runserver to start!
